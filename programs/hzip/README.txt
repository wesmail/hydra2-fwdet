Build hzip executable for zipping root files into zip files.
For documentation see base/util/hzip.cc. The executable gives
command line help  ./hzip -h


BUILD:

1. setup HYDRA  
2. make clean build install

REMARK:

Needs ROOT 5.28 to work correctly!
Since it depends on ROOT only it is best 
copied to $ROOTSYS/bin to be ready to
use.
